# Commercial Project Showcase (Redacted)

This repository is intentionally reduced to a small set of files that demonstrate engineering understanding only.

The rest of the commercial project was deliberately removed to protect:
- Client privacy
- Internal business logic
- Implementation details that could be directly reused

## What Is Included

Only the following core showcase files are kept:
- `showcase/01-architecture.md`: concise architecture snapshot
- `showcase/02-middleware-i18n-referral.ts`: middleware sample for i18n and referral cookie handling
- `showcase/03-auth-flow.ts`: simplified authentication flow (credentials + token fallback)
- `showcase/04-api-access-control.ts`: API route pattern with authorization and ownership checks
- `showcase/05-contact-form.tsx`: React form example with state and API submission
- `showcase/06-data-model.prisma`: partial data model sample (Users/Projects/Phases/Comments)
- `showcase/07-i18n-config.ts`: locale configuration and validation

## Important Notes

- This is not a full runnable product.
- Secrets, production configuration, and private operational code were removed.
- The purpose of this repo is technical review of approach and coding style, not delivery of commercial source code.

هذه النسخة مخصصة لعرض مستوى الفهم البرمجي فقط.

تم حذف بقية ملفات المشروع التجاري عمدًا لحماية:
