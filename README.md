# Commercial Project Showcase (Redacted)

هذه النسخة مخصصة لعرض مستوى الفهم البرمجي فقط.

تم حذف بقية ملفات المشروع التجاري عمدًا لحماية:
- خصوصية العميل
- منطق العمل الداخلي
- تفاصيل التنفيذ التي يمكن إعادة استخدامها مباشرة

## ما الذي بقي في المستودع؟

هذه الملفات الأساسية فقط لعرض المهارات:
- `showcase/01-architecture.md`: تصور معماري مختصر لتقسيم النظام.
- `showcase/02-middleware-i18n-referral.ts`: مثال Middleware للـ i18n وإدارة referral cookie.
- `showcase/03-auth-flow.ts`: مثال تدفق مصادقة (Credentials + token fallback) بشكل مبسط.
- `showcase/04-api-access-control.ts`: مثال Route مع تحقق صلاحيات وملف مرفق.
- `showcase/05-contact-form.tsx`: مثال React form مع إدارة state وطلبات API.
- `showcase/06-data-model.prisma`: جزء من نمذجة البيانات (Users/Projects/Phases/Comments).
- `showcase/07-i18n-config.ts`: إعداد اللغات والـ locale validation.

## ملاحظات مهمة

- هذه النسخة غير قابلة للتشغيل كمنتج كامل.
- أي أسرار/بيانات تشغيل/ملفات بنية إنتاجية تمت إزالتها.
- الهدف من النسخة: مراجعة التفكير الهندسي وجودة الأسلوب فقط، وليس تسليم الكود التجاري.
